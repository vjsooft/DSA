Three differance type of method

  • Constructor

    The constructor method is a special method of a class for creating and initializing an object instance of that class.

    Why to use it

    if we are assign the property inside a class or assign the value or property then use constructor

  • Prototype

    object prototype is a mechanism that allows objects to inherit properties and methods from other objects. Every JavaScript object has a prototype property that references another object, which is used as a template to create the inherited properties and methods.

    Why Use It:

    • Inheritance: It allows objects to inherit features from other objects, enabling code reuse and the creation of hierarchical structures.
    • Memory Efficiency: Shared methods are stored in the prototype, so multiple objects can use the same method without each object needing its own copy, saving memory.
    • Dynamic Behavior: Prototypes allow you to add or change properties and methods for all instances created from the prototype, enhancing flexibility and maintainability.
